Description

Now is the time to make the app usable so when the new user signs up and then logs in, there is a choice to deposit or withdraw money and view the balance..

Objectives

  • New user needs to create a new account at sign up form with all input data provided. If there is an empty input field, and the user pressed the "SIGN UP" button, then the app needs to return a warning with the message "Please input a missing field + name of the empty field". All input fields need to be stored at the press of the SIGN UP button so that the app could later access them.
  • If the sign up was successful, the app needs to return the message "Sign up successful" and return to the main screen. 
  • Then at the LOG IN screen user log in with username and password provided at sign up.
  • If user enters wrong username or password, the app must return a message with a "Wrong username or password" message.
  • If log in was successful, app redirects to account screen:

 Account screen with options :

  • deposit funds button  redirects to deposit funds screen ,
  • withdraw funds button redirects to withdraw funds screen,
  • view balance button redirects to view balance screen.
