Description

Because users will be accessing a bank account through the application, it needs to be secure as possible. We need to add an extra layer of protection with a safe code send to mobile.

Objectives

  • When the user wants to log in with the correct username and password, application should generate a random  4 - digit security code, that needs to be sent to the user's phone with a notification. User needs to enter that code to be able to log in successfully.
